Put your Zoom Spy source images here.

One image per question. Name them:
  q1.jpg   (or .png, .jpeg, .webp)
  q2.jpg
  q3.jpg
  q4.jpg
  q5.jpg
  q6.jpg

Any size or aspect ratio is fine. The script will create 5 zoomed stages from each.

Then run (from the server folder):
  npm run generate-zoom-spy

Output goes to q1/, q2/, ... as stage1.jpg (most zoomed) through stage5.jpg (full image).
